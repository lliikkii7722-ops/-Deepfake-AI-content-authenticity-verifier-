What this prototype does

Shows the UX and decision flow of a deepfake detector: ingest, preview, score, explain, and export.

What it does not do yet

It does not call production ML models. The scores are generated from deterministic browser-side heuristics for demo use.

Why this is still useful

You can publish it immediately, demo the product clearly, and later replace the scoring logic with real APIs or on-device models.

Ready to publish

Put it online as a static site.

Because this prototype is a single static website, publishing is simple. Upload the files to a static host and it will work immediately.

Netlify

Drag the project folder into Netlify Drop, or connect a Git repository. The `index.html` file becomes your home page automatically.

Vercel

Import the folder or repository as a static project. No framework settings are required for this version.

GitHub Pages

Push the files to a repository and enable Pages on the root branch. This works well if you want a free public demo link.

Current deliverables

`index.html` contains the full working prototype, and `README.md` explains fast publishing steps.
